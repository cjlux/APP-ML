\section*{Acquis d'apprentissage visés}

À la fin de cet apprentissage par problème de 3 $\times$ 3 h, chaque étudiant saura :
\begin{itemize}
\item[$\rhd$] Programmer en langage Python un {\em Réseau de Neurones Dense} (RND) dédié à la classification de données
  en utilisant les modules \boldtt{tensorflow2} et \boldtt{keras}.
\item[$\rhd$] Expliquer la structure des données labellisées construites avec les données temporelles brutes issues du banc de perçage.
\item[$\rhd$] Conduire l'entraînement supervisé du RND en utilisant les données labellisées disponibles dans un fichier CSV.
\item[$\rhd$] Évaluer les performances du réseau entraîné en utilisant des données de test, en particulier afficher la matrice de confusion.
\end{itemize}

\begin {bclogo}[noborder=true, couleurBarre=Chocolate, logo=\bctrombone]{}
\vspace*{-4mm}\textbf{\large Créer et configurer l'Environnement Virtuel Python \DarkRed{\boldtt{minfo\_ml}}}\\[1mm]
Sur ton ordinateur portable avec \boldtt{miniconda3} déjà installé:
\begin{itemize}
    \item Ouvre une console \boldtt{Anaconda Prompt} (Windows) ou un \boldtt{terminal} (Mac ou Linux).
    \item Créé l'EVP  \DarkRed{\boldtt{minfo\_ml}} en tapant la commande : \commandBF{conda create -n minfo\_ml pip python=3.8}\\
  puis réponds aux questions pour télécharger et installer les paquets Python...
    \item Active l'EVP \DarkRed{\boldtt{minfo\_ml}} en tapant la commande : \commandBF{conda activate minfo\_ml}
  \end{itemize}

Les modules nécessaires à l'APP sont ensuite installés en utilisant le fichier \fileBF{requirements.txt} :\\[-6mm]
\begin{itemize}
  \item Positionne-toi dans le dossier \fileBF{APP-ML} avec la commande :\\
(Windows) $\leadsto$ \commandBF{cd C:\bsh<chemin\_du\_dossier\_APP-ML\_copié\_depuis\_le\_navigateur\_de\_fichiers>}\\
(Mac/Linux) $\leadsto$ \commandBF{cd /home/$\cdots$/APP-ML}
  \item Charge les modules Pythons nécessaires en tapant : \commandBF{pip install -r requirements.txt}\\
  Les modules et leurs dépendances sont téléchargés et installés...
  \item Complète l'installation avec la commande : \commandBF{conda install numpy pydot pydotplus}
\end{itemize}
\end{bclogo}

\section*{\ding{220} Séance 1 \& 2 -- Auto-formation, prise en main du problème}

\begin {bclogo}[noborder=true, couleurBarre=Chocolate, logo=\bctrombone]{}
  \vspace*{-4mm}\textbf{Auto-formation: travail personnel avec les {\em notebooks} de l'APP}\\[-2mm]

  Les trois {\em notebooks} de l'APP permettent d'acquérir les savoir-faire nécessaires pour construire un réseau de neurones dense
  et apprendre à l'entraîner avec des données labellisées.

Chaque étudiant doit s'auto-former en travaillant les {\em notebooks} chargés dans un processus {\em jupyter notebook} lancé dans l'EVP \DarkRed{\boldtt{minfo\_ml}}.
\end{bclogo}  

\begin {bclogo}[noborder=true, couleurBarre=Chocolate, logo=\bctrombone]{}
  \vspace*{-4mm}\textbf{Problème à traiter en équipe : entraîner un RND à classifier des données labellisées issues d'un banc de perçage}
  
Le travail se fait avec le notebook \file{ML-drilling.ipynb} du dossier \file{Drilling-Problem}. Le plan de travail poposé est :
\begin{enumerate}
\item Comprendre comment est constitué le jeu de données labellisées qui va servir à entraîner le réseau de neurones.
\item Construire le RND en utilisant les acquis d'apprentissage de l'auto-formation.
\item Conduire l'entraînement supervisé du RND avec le jeu des données labellisées.
\item Évaluer les performances du RND entraîné en utilisant les données de test.
\item Rechercher parmi les données d'entraînement celles qui sont le plus pertinentes.
\end{enumerate}

Le projet est conduit par l'équipe qui doit répartir entre ses membres les tâches à réaliser pour le projet.
\end{bclogo}
